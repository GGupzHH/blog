# 在`GitHub`搭建自己的主页

### 设置
:::tip
这里其实就和自己去服务器上部署一样，只不过这里是指定目录把需要部署的页面放进去即可，并且建议是打包之后的文件。
:::

  ![流程](/blog/images/document-github/7431675827239_.pic.jpg)
  1. 进入到自己想要部署的项目中
  2. 从`settings`进入
  3. 找到左侧`Pages`
  4. 选择要部署的分支
  5. 选择文件夹目录
     - 根目录
     - /docs
  6. 最后保存即可
